# Game Over Videos

Place your game over videos here. The game will randomly pick one to play when the player dies.

## Required Files

Upload 1-5 video files with these exact names:
- `gameover1.mp4`
- `gameover2.mp4`
- `gameover3.mp4`
- `gameover4.mp4`
- `gameover5.mp4`

You don't need all 5 - the game will only use the ones that exist.

## Recommended Format

- **Format:** MP4 (H.264 codec)
- **Resolution:** 720p or lower (for faster loading)
- **Duration:** 3-10 seconds (loops automatically)
- **File size:** Under 5MB each for best performance

## How It Works

1. When the player dies, the game randomly selects one of the available videos
2. The video plays on loop in the game over screen
3. A different video may be selected each time you die
