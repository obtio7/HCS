# Sound Effects

Drop your sound files here.

## Required Files

- `bgm.mp3` — Background music during gameplay (loops)
- `gameover-bgm.mp3` — Background music on game over screen (loops)

## Optional Sound Effects

- `jump.mp3` — Played when the character jumps
- `crash.mp3` — Played on collision/game over
- `milestone.mp3` — Played every 100 points

If any sound file is missing, that sound will be silently disabled.

## Recommended Format

- **Format:** MP3
- **Bitrate:** 128-192 kbps
- **Sample rate:** 44.1 kHz
